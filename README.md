## рҹ§  How It Works

1. User logs in using the demo credentials.
2. The website loads available vocabulary lessons from an API.
3. When a lesson is selected, vocabulary cards appear.
4. Users can:
   - View word meanings
   - See example sentences
   - Check synonyms
   - Hear pronunciation
5. Users can also search for vocabulary using the search bar.
